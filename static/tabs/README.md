# TabBar图标目录

此目录存放TabBar图标资源。

## 图标要求

| 图标 | 尺寸 | 格式 | 说明 |
|------|------|------|------|
| project.png | 81×81 px | PNG | 项目Tab - 默认 |
| project-active.png | 81×81 px | PNG | 项目Tab - 选中 |
| program.png | 81×81 px | PNG | 编程Tab - 默认 |
| program-active.png | 81×81 px | PNG | 编程Tab - 选中 |
| monitor.png | 81×81 px | PNG | 监控Tab - 默认 |
| monitor-active.png | 81×81 px | PNG | 监控Tab - 选中 |
| tools.png | 81×81 px | PNG | 工具Tab - 默认 |
| tools-active.png | 81×81 px | PNG | 工具Tab - 选中 |
| profile.png | 81×81 px | PNG | 我的Tab - 默认 |
| profile-active.png | 81×81 px | PNG | 我的Tab - 选中 |

## 临时解决方案

开发阶段可使用Unicode字符代替：
- 项目: 📁
- 编程: 🔧
- 监控: 📊
- 工具: 🧮
- 我的: 👤

## 配色建议

图标使用青色(#00BCD4)作为激活态主色，灰色(#607D8B)作为默认态。
建议使用扁平化设计，与整体UI风格保持一致。
